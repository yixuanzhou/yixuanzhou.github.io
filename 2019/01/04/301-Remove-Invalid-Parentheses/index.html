301. Remove Invalid Parentheses

Posted on 2019-01-04 | In LeetCode

Problem Description:

Remove the minimum number of invalid parentheses in order to make the input string valid. Return all possible results.

Note: The input string may contain letters other than the parentheses ( and ).

Example 1:

Input: "()())()"
Output: ["()()()", "(())()"]

Example 2:

Input: "(a)())()"
Output: ["(a)()()", "(a())()"]

Example 3:

Input: ")("
Output: [""]

Note

BFS. Use HashSet and LinkedList to store all possible substrings. And check if each substring is valid parentheses.

Code

public List<String> removeInvalidParentheses(String s) {

    // s = s.replaceAll("[^().]", "");
    List<String> res = new ArrayList<>();
    Queue<String> waitlist = new LinkedList<>();
    Set<String> visited = new HashSet<>();
    waitlist.add(s); visited.add(s);

    while (!waitlist.isEmpty()) {
        String cur = waitlist.poll();
        if (isValidParentheses(cur)) {
            if (!res.isEmpty() && cur.length() < res.get(0).length()) continue;
            res.add(cur); continue;
        }

        for (int i = 0; i < cur.length(); i++) {
            if (cur.charAt(i) != '(' && cur.charAt(i) != ')') continue;
            String sub = cur.substring(0, i) + cur.substring(i+1);
            if (visited.contains(sub)) continue;
            visited.add(sub); waitlist.add(sub);
        }
    }

    return res;
}

private boolean isValidParentheses(String s) {
    int count = 0;

    for (int i = 0; i < s.length(); i++) {
        char c = s.charAt(i);
        if (c == '(') count++;
        if (c == ')' && count-- == 0) return false;
    }

    return count == 0;        
}
